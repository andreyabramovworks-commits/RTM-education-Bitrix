# Версии RTM Education

## v50 — Excalidraw, тесты и навигация курса

- просмотр фреймов точно подгоняется по границам и не уменьшается ниже базового масштаба;
- редактор принимает Excalidraw clipboard JSON вместе с элементами, связями, текстом, изображениями и стилями;
- исправлены шрифты, переключатели подчёркивания и зачёркивания, а также карточка завершения;
- стрелочная навигация ограничена материалами одного курса и учитывает секции и обязательность;
- новый движок тестов поддерживает один или несколько ответов, соответствия, точный и свободный текст и результаты;
- запрос `tasks.task.add` кодирует вложенные поля по формату Bitrix24 и возвращает точную ошибку портала вместо безусловного 502.

## v49.2.6 — роли внутри мобильного меню

## Откат production

Откат выполняется осознанно: сначала нужный тег переносится в `main`, после
чего обычный автодеплой развернёт его на сервере.

```bash
git fetch origin --tags
git switch main
git pull --ff-only
git revert --no-commit v47.0.0..main
git commit -m "rollback: restore v47.0.0"
git push origin main
```

Перед откатом базы необходимо проверить совместимость миграций. Резервная
копия PostgreSQL создаётся отдельно от Git и исходников приложения.
