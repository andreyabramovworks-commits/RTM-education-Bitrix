# Версии RTM Education

Production-сервер автоматически разворачивает только последнюю проверенную
версию из ветки `main`. Стабильные снимки приложения публикуются отдельными
Git-тегами, поэтому их можно скачать или использовать для отката независимо от
дальнейшей разработки.

## v47.0.0 — стабильный интерфейс v46

- исходный визуальный интерфейс v46 сохранён без React-переписывания;
- данные проектов, курсов, статей и тестов синхронизируются с PostgreSQL;
- Excalidraw-сцены сохраняются на сервере и доступны другим сотрудникам;
- материалы, расположенные непосредственно в проекте, поддерживаются наравне с
  материалами внутри курса.

Скачать исходники можно на странице GitHub `Tags`, выбрав `v47.0.0` и архив
`zip` или `tar.gz`.

## v48.0.0 — React

Версия будет опубликована после переноса интерфейса в React и визуального
сравнения с v47. До публикации тега незавершённая v48 не считается стабильной.

## Откат production

Откат выполняется осознанно: сначала нужный тег переносится в `main`, после
чего обычный автодеплой развернёт его на сервере.

```bash
git fetch origin --tags
git switch main
git pull --ff-only
git revert --no-commit v47.0.0..main
git commit -m "rollback: restore v47.0.0"
git push origin main
```

Перед откатом базы необходимо проверить совместимость миграций. Резервная
копия PostgreSQL создаётся отдельно от Git и исходников приложения.
